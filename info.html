A better way to edit.

About

Vector is a free GIS viewer and editing tool. It is intended for GIS users who create and manage their data in Esri's ArcServer and/or ArcGIS Online. If you use public or private Feature Services, Map Services, or Hosted Feature Services this tool is for you.

Vector exists not to replace fully-featured desktop GIS software. Instead, it is intended to be an alternative experience for viewing and editing your data over the web. It provides a simple interface to create and edit both geometry and attribute data. Vector does not have the complete editing tools of desktop software, but it is a great option for most editing situations.

Vector was built using the open source libraries of Leaflet, Esri Leaflet, and Leaflet Geometry Management. These are all awesome projects. A big thank you goes out to the creators and maintainers of them. Steve Cline created Vector, if you use it and like, feel free to let him know.

Help

Connecting to your Services

Vector is designed for use with secure (https) Esri Feature Services, Map Services and Hosted Feature Services. To connect to a layer in your service, either public or private, you must first enter a valid service layer URL, including the array based number at the end. This is done through the 'Add Service' button. Services over http are not allowed with Vector.

Example Service URL
https://sampleserver6.arcgisonline.com/arcgis/rest/services/Hurricanes/MapServer/1

Authentication

Vector uses the ArcGIS REST API to do token based authentication. If your service requires authentication, you will be prompted to provide a Token URL to your server and your user name and password credentials. Vector sets all tokens to expire in 12 hours. If you wish to connect to a Hosted Feature Layer from ArcGIS Online, click the checkbox and it will populate the appropriate Token URL from below. Provide a user name and password for an authorized Named User of the service from your ArcGIS Online account.

Example Server Token URL
https://sampleserver6.arcgisonline.com/arcgis/tokens/generateToken

ArcGIS Online Token URL
https://www.arcgis.com/sharing/rest/generateToken

Editing Capabilities

When you create a Feature Service or Map Service, you must configure which editing capabilites you would like. If no editing capabilites are set, you may still add the service layer to Vector but it will be view only. As a view only service layer, you will still be able to style and view feature attributes, but you will not be able to edit the data. If you would like to test a view only service layer, refer to the example Service URL above.

Editing Tips

To create a new feature, hover over the three vertical dots that are located right of the layer name in the sidebar. Click the top option with the pencil icon. For point data, click on the map once, to place a new point. For line and polygon data, click once to draw vertices of the new feature and double click to complete drawing. Alternatively, you may also single click on the first vertex of a polygon feature or the last vertex of a line feature to the complete drawing. This second method is recommended.

To edit a feature, click the feature on the map to select it. Then click the 'Edit' button in the layer attribute panel. Right-click a vertex to delete it. To create a new vertex, click and drag the false vertex midpoint between two vertices. To save edits, click the 'Save' button. To abandon edits, click the '×' in the attribute panel. This will also unselect the feature and close the panel.

Snapping

Snapping functionality is turned on by default. To temporarily disable snapping, hold down 'Alt' on your keyboard.

Browser Support

Vector is designed to work in modern browsers such as Chrome, Firefox and Edge on desktop computers. It is not advised to use Vector on tablets, smart phones, or touch screen devices. A computer mouse offers a better editing experience than a finger.
